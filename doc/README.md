## How It Works (Step-by-Step)

1. **Admin creates enquiry** → Enters farmer details
2. **Admin sends form link** → Via email or WhatsApp
3. **Farmer receives link** → Clicks to open form
4. **Farmer fills form** → 4-5 questions, 5-10 minutes
5. **Real-time calculation** → Shows govt subsidy amount
6. **Farmer uploads docs** → Aadhar, land cert, etc
7. **Farmer submits** → Gets reference number
8. **Auto-DPR generated** → Government-format document
9. **Admin downloads all** → ZIP file with everything
10. **Farmer tracks status** → Via reference number
